# Contributing

Contributions are welcome — issues, bug reports, and pull requests are all
appreciated.

A few light suggestions:

- For non-trivial changes, opening an issue first to discuss the approach
  saves everyone time.
- Please make sure the existing DSP unit tests still pass (`ctest` in the
  `cpp/build-tests` directory after configuring with `-DMONKSYNTH_BUILD_TESTS=ON`).
- Keep PRs focused — one logical change per PR where practical.

Build instructions are in the README. If something isn't clear, ask.
